# ⌨️ Keyboard Shortcut Cheatsheet

A comprehensive reference page for common keyboard shortcuts on Windows and Mac to boost your productivity.

## ✨ Features

- **Windows & Mac Support** - Toggle between Windows and Mac shortcuts with one click
- **8 Categories per OS**:
  - 💻 General
  - 🧭 Navigation
  - 📸 Screenshot
  - 🌐 Browser
  - ✏️ Text Editing
  - 📁 File Management
  - ♿ Accessibility
  - 🖥️ Virtual Desktops / Mission Control
- **100+ Shortcuts** - Comprehensive coverage of essential shortcuts
- **Real-time Search** - Search by description or key combination
- **Category Filtering** - Browse shortcuts by category
- **Click to Copy** - Click any shortcut to copy it to clipboard
- **Print-Friendly** - Optimized CSS for printing your cheatsheet
- **Responsive Design** - Works beautifully on desktop, tablet, and mobile
- **Modern Dark UI** - Beautiful keyboard-style key caps

## 🚀 How to Run

1. Open `index.html` in your web browser
2. That's it! No build tools or dependencies required

## 🎯 How to Use

- **Toggle OS**: Click Windows or Mac buttons to switch shortcut sets
- **Search**: Type in the search bar to find shortcuts by description or keys
- **Filter by Category**: Click category tabs to view specific shortcut types
- **Copy Shortcut**: Click any shortcut to copy it to your clipboard
- **Print**: Click the 🖨️ Print button to get a printer-friendly version
- **Browse**: Scroll through all categories to discover new shortcuts

## 📋 Shortcut Categories

### Windows
- **General**: Copy, paste, undo, save, and more essentials
- **Navigation**: Open Start, Settings, Search, Task View
- **Screenshot**: PrtScn, Snipping Tool, Game Bar
- **Browser**: Tabs, zoom, bookmarks, history
- **Text Editing**: Bold, italic, alignment, word navigation
- **File Management**: Explorer, rename, delete, properties
- **Accessibility**: Magnifier, Narrator, Sticky Keys
- **Virtual Desktops**: Create, switch, and close desktops

### Mac
- **General**: ⌘ copy, paste, undo, quit, Spotlight
- **Navigation**: App switching, Mission Control, Finder
- **Screenshot**: ⌘ Shift 3/4/5 for different capture modes
- **Browser**: Safari/Chrome tab and navigation shortcuts
- **Text Editing**: ⌘ formatting, word/line navigation
- **File Management**: Finder, Trash, folder navigation
- **Accessibility**: VoiceOver, Zoom, Full Keyboard Access
- **Mission Control**: Desktop switching and app exposé

## 🖨️ Printing

The cheatsheet is optimized for printing:
- Click the **Print** button or press `Ctrl+P` / `⌘+P`
- Unnecessary UI elements are hidden in print mode
- Clean black-and-white layout for paper saving
