# 😂 Meme Viewer

Browse and display popular memes from a static collection with a beautiful, modern UI.

## ✨ Features

- **30 Popular Memes** - Curated collection of classic, animal, programming, reaction, and motivational memes
- **Search Functionality** - Quickly find memes by title or description
- **Category Filtering** - Filter memes by category:
  - 🎭 Classic
  - 🐾 Animals
  - 💻 Programming
  - 😂 Reactions
  - 💪 Motivational
- **Favorites System** - Save your favorite memes to localStorage
- **Modal View** - Click any meme to see full details with description
- **Copy Link** - Copy meme image links to clipboard
- **Download** - Download memes directly
- **Pagination** - Browse memes in pages (12 per page)
- **Live Stats** - View total memes, showing count, and favorites
- **Responsive Design** - Works beautifully on desktop, tablet, and mobile
- **Dark Theme** - Modern dark UI with smooth animations

## 🚀 How to Run

1. Open `index.html` in your web browser
2. That's it! No build tools or dependencies required

```
meme-viewer/
```

## 🎯 How to Use

- **Search**: Type in the search bar to filter memes by title or description
- **Filter**: Use the dropdown to select a specific category
- **Browse**: Click on any meme card to view it in full detail
- **Favorite**: Click the heart icon on any card or in the modal
- **Navigate**: Use Previous/Next buttons to browse through pages
- **Download**: Click the download button in the modal view
- **Copy Link**: Click the copy link button to share memes

## 🛠️ Built With

- **HTML5** - Semantic markup with accessibility attributes
- **CSS3** - CSS variables, flexbox, grid, animations
- **JavaScript (ES6+)** - Modern JavaScript with localStorage

## 📱 Responsive Breakpoints

- Desktop: 1400px max-width container
- Tablet: 768px and below
- Mobile: 480px and below

## 💾 Local Storage

Favorites are persisted in `localStorage` under the key `memeFavorites`.
