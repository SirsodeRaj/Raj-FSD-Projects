# ✨ Motivational Wallpaper Page

Display an inspirational background wallpaper with a daily motivational quote to inspire and uplift your spirit.

## ✨ Features

- **Daily Wallpaper** - Automatically changes wallpaper each day based on the date
- **30 Unique Wallpapers** - Beautiful gradient backgrounds with inspirational quotes
- **5 Categories**:
  - 🏆 Success
  - 🌱 Life
  - 💪 Courage
  - 🧠 Wisdom
  - 😊 Happiness
- **Category Filtering** - Browse wallpapers by category
- **Favorites System** - Save your favorite wallpapers to localStorage
- **One-Click Refresh** - Get a random new wallpaper instantly
- **Download Wallpapers** - Download wallpapers as 1920x1080 PNG images with quote
- **Full-Screen Background** - Current wallpaper displays as full-screen background
- **Responsive Design** - Beautiful on desktop, tablet, and mobile
- **Smooth Animations** - Elegant transitions and fade effects

## 🚀 How to Run

1. Open `index.html` in your web browser
2. That's it! No build tools or dependencies required

## 📁 Project Structure

```
motivational-wallpaper/
├── index.html      # Main HTML with semantic structure
```

## 🎯 How to Use

- **View Daily Wallpaper**: The homepage shows a new inspirational wallpaper each day
- **Refresh**: Click the 🔄 button to get a random wallpaper
- **Browse**: Scroll down to browse all wallpapers in the gallery
- **Filter**: Click category tabs to filter wallpapers by theme
- **Set as Background**: Click any wallpaper card to set it as the full-screen background
- **Favorite**: Click the heart icon to save wallpapers to your favorites
- **View Favorites**: Click the ❤️ button to see all your saved wallpapers
- **Download**: Click the 💾 button to download the current wallpaper as a PNG image

## 🎨 Wallpaper Specifications

- **Resolution**: Generated downloads are 1920x1080 pixels (Full HD)
- **Format**: PNG with gradient backgrounds
- **Quotes**: 30 curated motivational quotes from famous authors and leaders
- **Categories**: Success, Life, Courage, Wisdom, Happiness

## 🛠️ Built With

- **HTML5** - Semantic markup with accessibility attributes
- **CSS3** - CSS variables, gradients, flexbox, grid, animations, backdrop filters
- **JavaScript (ES6+)** - Modern JavaScript with Canvas API for image generation and localStorage

## 💾 Local Storage

Favorites are persisted in `localStorage` under the key `wallpaperFavorites`.
