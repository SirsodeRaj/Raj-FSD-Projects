# 🚀 Getting Started

## 📥 Clone The Repository

```bash
git clone https://github.com/SirsodeRaj/Raj-FSD-Projects.git
```

---

## 📂 Move Into Project Directory

```bash
cd Raj-FSD-Projects
```

---

## 📦 Install Dependencies

```bash
npm install
```

or

```bash
npm i
```

---

## ▶️ Start Development Server

```bash
npm run dev
```

---

## 🌐 Open In Browser

```bash
http://localhost:5173
```
